# Android & Kotlin – Fundamentos

Repositório dedicado ao estudo prático de Kotlin e desenvolvimento Android,
com foco em construção de base sólida para projetos mobile.

## Objetivo
Consolidar fundamentos de Kotlin, lógica de programação e estrutura de aplicativos Android,
visando evolução acadêmica e preparação para atuação profissional.

## Conteúdo (em progresso)
- Fundamentos da linguagem Kotlin
- Estrutura básica de projetos Android
- Exercícios práticos
- Pequenos projetos de estudo

## Status
Repositório em desenvolvimento contínuo.
